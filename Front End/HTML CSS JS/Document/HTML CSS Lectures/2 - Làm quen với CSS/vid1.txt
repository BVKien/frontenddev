+ Sử dụng CSS trong HTML
- 3 cách 
- internal
- external 
- inline

-- Internal

<style>
h1 {
    color: red;
}
</style>
</head>

-- External

- tạo 1 thẻ link ddeebs ddunowgf dẫn file.css

-- Inline
